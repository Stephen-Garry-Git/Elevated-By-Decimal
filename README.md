# Elevated By Decimal

A modern business website developed for Elevated By Decimal, designed to establish a strong online presence and communicate the brand's services in a clean, professional, and engaging way.

🌐 Live Website: https://www.elevatedbydecimal.com/

---

## Project Overview

Elevated By Decimal is a business-focused website created to showcase the company's services, brand identity, and value proposition through a modern and responsive web experience.

This repository contains the version of the project that I worked on during the early development phase before the website evolved into later frameworks and technologies.
